# DOOM-Style Raycaster

A simple raycasting engine similar to early DOOM games.

## Controls
- **WASD** - Move
- **Arrow Keys** - Rotate view
- **ESC** - Quit

## Building on GitHub
This project automatically builds on GitHub Actions. Check the Actions tab for build artifacts.

## Local Build (if you have SDL2 installed)
```bash
make
./raycaster
